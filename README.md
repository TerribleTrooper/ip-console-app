# IP Console App

Консольное приложение, которое:

- читает IP-адреса из файла  
- получает информацию по каждому IP с сервиса **ipinfo.io**  
- группирует IP-адреса по странам  
- определяет страну с наибольшим количеством IP  
- выводит список городов для этой страны  

---

## Используемые технологии

- .NET
- HttpClient
- Newtonsoft.Json
- LINQ

---

## Как работает приложение

1. IP-адреса считываются из текстового файла (один IP на строку)
2. Для каждого IP выполняется HTTP-запрос к `https://ipinfo.io/{ip}/json`
3. Полученные данные агрегируются в памяти
4. Выполняется группировка IP по странам
5. Определяется страна с максимальным количеством IP-адресов
6. В консоль выводятся города, относящиеся к этой стране

- Все HTTP-запросы выполняются асинхронно

- Все вычисления выполняются без использования базы данных
